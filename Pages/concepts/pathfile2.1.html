상대 경로 사용 보기

현재 경로를 보고, 그 내용을 보는 예제: 

   pwd (현재 경로 이름 보기)
   /home/eucs/jane (현재 경로의 절대 경로 이름)
   ls -l presentation/docs/oview.html (파일 정보 보기)

윗 예제는 현재 경로 안에는 presentation 이라는 하위 경로의 docs 라는 하위 경로의 oview.html 파일의 정보를 보는 명령입니다.

다른 경로로 옮겨가기 예제:

   pwd (현재 경로 이름 보기)
   /home/eucs/jane/presentation/docs (절대 경로 이름)
   cd ../illustrations (상대 경로 사용)
   pwd (현재 경로 이름을 확인)
   /home/eucs/jane/presentation/illustrations

윗 예제는 jane 경로 이름을 상대 경로로 사용한 것입니다.

경로 이름에 ~ 문자 사용하기:

   cp ~fran/tmp/for_mike.txt

윗 예제는 for_mike.txt 파일을 fran 이라는 사용자의 홈 디렉토리 안에 있는 tmp 경로안으로 복사하는 것입니다. 이때, ~ 문자는 바로 fran 이라는 사람의 홈 디렉토리의 절대 경로로 대치됩니다.
