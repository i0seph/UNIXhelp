쉘에서의 사용되는 특별한 의미의 문자들

쉘에서 사용되는 메타문자는 다음과 같습니다:

   ;  &  ( )  |  ^  <  >  ~ %  { }
   $  #   '  "  \  @@  *  ?  [ ]  -  !
   Newline (a <Return>)
   Space   (a <Space>)
   Tab	   (a <Tab>)

쉘에서, 이들 문자들을 그 문자열 자체를 뜻하는 것이 아니고, 다른 특별한 의미를 지니고 있습니다.

이런 문자들을 표현하기 위해서 따움표로 묶어 명령어의 인자로 표현하기도 하고, 정규 표현식으로 사용되기도 합니다.

  • 이 특별한 의미를 없애기
