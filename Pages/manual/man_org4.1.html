온라인 설명서에서 사용하는 표제들

하나의 온라인 설명서는 다음과 같은 여러 부분으로 나뉘어져 있습니다. 물론 모든 설명서가 아래의 모든 부분을 다 포함하고 있지는 않습니다. 하지만, NAME, SYNOPSIS, DESCRIPTION 부분은 대부분의 설명서에 포함되어 있습니다.

해당 이름을 선택하면 그에 대한 간단한 설명을 보여줍니다.

  • NAME
  • SYNOPSIS
  • DESCRIPTION
  • OPTIONS
  • ENVIRONMENT
  • EXAMPLES
  • FILES
  • SEE ALSO
  • WARNINGS
  • DIAGNOSTICS
  • BUGS
