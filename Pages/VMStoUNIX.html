VMS 사용자를 위한 UNIX 설명

설명                        VMS                     UNIX


파일 목록 보기              DIRECTORY               ls
                            DIRECTORY/FULL          ls -l
파일 내용 보기              TYPE                    cat
파일내용 화면 단위로 보기   TYPE/PAGE               more, less
파일의 처음 몇 줄 보기                              head
파일의 끝 몇 줄 보기                                tail
파일 복사                   COPY                    cp
파일 안의 문자열 찾기       SEARCH                  grep
                                                    fgrep
파일 비교                   DIFF                    diff
                                                    cmp
파일 이름 바꾸기            RENAME                  mv
파일이나 경로 지우기        DELETE                  rm
                                                    rmdir
파일 속성 바꾸기            SET FILE/PROT           chmod
파일 소유주 바꾸기          SET FILE/OWNER          chown
                                                    chgrp
경로 만들기                 CREATE/DIR              mkdir
작업 경로 바꾸기            SET DEFAULT             cd
현재 작업 경로 보기         SHOW DEFAULT            pwd
도움말 보기                 HELP                    man
                                                    apropos
날짜와 시간 보기            SHOW TIME               date
디스크 남은 공간 보기       SHOW DEV                df
프로세스 멈추기             STOP                    kill
링크                        LINK                    ld
인쇄                        PRINT                   lpr
프린트 큐 보기              SHOW QUEUE              lpq
프린트 종류 보기            SHOW ENTRY              lpq
패스워드 바꾸기             SET PASSWORD            passwd
현재 사용자 보기            SHOW USERS              who
                                                    finger
                                                    w
프로세스 보기               SHOW PROCESS            ps
터미날 설정                 SET TERMINAL            stty
다른 사용자와 이야기하기    PHONE                   talk
메시지 수신 거부            SET NOBROADCAST         mesg n
