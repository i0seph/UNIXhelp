command history 사용하기

쉘 명령행에서 입력하고 실행한 각 명령 내용들은 command history list에 기록됩니다. 이것은 이전에 사용했던 명령을 다시 입력하는 불편함을 덜기 위해서 제공하는 기능으로 사용자는 이 목록의 명령 - 이전에 자기가 실행했던 명령 - 보다 빠르게 다시 사용할 수 있습니다. 이것을 command history 기능이라고 합니다.

이 기능은 Bourne shell (sh)을 제외한 모든 쉘에서 제공하고 있습니다.

  • C shell의 command history
  • TC shell의 command history
  • Korn shell의 command history
  • Bourne Again SHell의 command history
