다음 login을 위해 command history 저장하기

Bourne Again 쉘에서는 자동으로 사용자가 현재 쉘을 마치게 되면, command history list 내용을 .bash_history 파일에 저장합니다.

그리고, 다음 login을 하면, .bash_history 파일을 읽어 그 내용을 command history list에 다시 불러와서, 이전 접속 때 사용했던 명령들을 다시 사용할 수 있게 합니다.

이때 저장될 크기를 HISTFILESIZE 환경 변수 값으로 조절할 수 있습니다. 예를 들어:

   HISTFILESIZE=25;export HISTFILESIZE

이렇게 하면, 최근 사용한 25개의 명령만 이 파일에 저장됩니다.
