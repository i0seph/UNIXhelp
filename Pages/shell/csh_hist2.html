다음 login을 위해 command history 저장하기

이전에 접속해서 사용했던 command history를 다음 접속 때도 사용하려면, .login 파일을 열어, savehist 쉘 변수의 값을 적당하게 지정해 줍니다. 예를 들어:

   set savehist=25

이렇게 하면, logout 할때, 25개의 이전 명령들을, .history 파일에 저장하고, 다음 login인 때, 이 파일을 읽어, history list에 저장해 둡니다. 이러면, 이전 login 때 사용했던 명령들을 사용할 수 있지요.

이 값을 너무 크게 잡으면, login 과정이 다소 늦어 질 수 있습니다. 또한 history에서 지정한 값 보다 더 크게 지정할 수는 없겠지요.
