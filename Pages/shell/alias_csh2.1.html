alias 인자 처리 보기

하나의 인자를 사용하는 경우:

   alias print 'lpr \!^ -Pps5'
   print memo.txt

!^ 기호는 하나의 첫번째 지정한 인자로 대치됩니다. 그래서, 윗 명령은 다음과 명령으로 변환되어 실행됩니다. 

   lpr memo.txt -Pps5

! 문자는 쉘에서 특별한 의미로 사용되는 문자이기 때문에, 이 기능을 금지하기 위해 \ 문자를 먼저 사용한 것에 주의하십시오.

여러 인자를 사용하는 경우:

   alias print 'lpr \!* -Pps5'
   print part1.ps glossary.ps figure.ps

!* 기호는 그 alias에서 지정한 모든 모든 인자로 대치됩니다. 그래서 윗 명령은 다음과 같은 명령으로 해석되어 실행됩니다. 

   lpr part1.ps glossary.ps figure.ps -Pps5

! 문자는 쉘에서 특별한 의미로 사용되는 문자이기 때문에, 이 기능을 금지하기 위해 \ 문자를 먼저 사용한 것에 주의하십시오.
