foreground 프로세스 보류하기 보기

예제:

   find / -name *.c -print
   find: cannot read /dev/diag
   ...
   ^Z
   [1]+  Stopped                 find / -name *.c -print
   ps
   22926 p9 S     0:03 -Tcsh (Tcsh)
   29857 p9 T     0:00 find / -name *.xdh -print
   29859 p9 R     0:00 ps
   mail tony < memo.txt

이것은 먼저 루트 디렉토리를 대상으로 .c 문자열을 포함하는 모든 파일을 찾기 명령을 내린 것입니다.

이 작업은 시스템 전체의 모든 파일을 검색하는 것임으로 엄청난 시간이 걸립니다. 이때, 사용자가 이것을 일단 보류하고,

현재 자신의 프로세스를 살펴 보고, tony라는 사용자에게 편지를 보내는 것입니다. 편지 보내기를 끝내면, 프로세스를 계속 진행하거나, 아니면, background job으로 바꿀 수 있겠지요.
