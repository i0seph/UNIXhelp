Korn shell의 filename completion

filename completion 기능을 사용하려면 EDITOR나 VISUAL 환경 변수 값이 emacs, gmacs, vi 중 하나로 지정되어 있어야 합니다. 

   EDITOR=emacs

이때, VISUAL 변수값도 지정되어 있다면, 이 값을 사용합니다. VISUAL 변수가 우선순위가 높음.

emacs- 또는 gmacs-스타일 file completion

사용법: ESC 글쇠를 두번 칩니다.

   ls -l on<ESC><ESC>line_help

이때 입력한 문자열로 시작하는 이름이 여러개 있을 경우에는 경고음만 냅니다.

그 이름이 디렉토리일 경우는 그 이름 끝에 / 문자가 나타납니다.

vi-스타일 file completion

사용법: \ 글쇠.

   ls -l on\line_help

이때 입력한 문자열로 시작하는 이름이 여러개 있을 경우에는 경고음만 냅니다.

그 이름이 디렉토리일 경우는 그 이름 끝에 / 문자가 나타납니다.

가능한 이름 목록

입력한 문자열로 시작하는 이름을 가진 모든 것을 보려면, ESC 글쇠를 치고, = 글쇠를 칩니다.

   ls -l S<ESC>=
   1) SIRS
   2) Solaris
