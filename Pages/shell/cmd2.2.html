Korn shell의 명령행 편집

Korn shell에서의 편집 명령은 EDITOR나, VISUAL 환경 변수 값에 따라 다르게 작동합니다. 여기서 사용할 수 있는 값은 emacs, gmacs, vi 셋 중 하나입니다.

편집 방법을 EMACS 스타일로 하려면, .profile 안에 다음 내용을 입력합니다. 

   EDITOR=emacs

이때, VISUAL 변수값도 지정되어 있다면, 이 값을 사용합니다. VISUAL 변수가 우선순위가 높음.

쉘 실행 중에 임시로 편집 모드를 바꾸려면, 다음과 같이 할 수 있습니다. 

   set -o vi

물론 이 내용을 .profile 파일안에 넣어 두고, 실행될 때 마다 지정할 수도 있습니다.

각 편집 모드는 각 편집기에서 사용하는 편집 방법을 그대로 사용합니다. 즉 emacs, gmacs 값이 지정되었다면, EMACS 편집기에서 사용하는 편집방법을 사용할 수 있고, vi 값이 지정되었다면, vi 편집기에서 사용하는 편집방법을 사용할 수 있습니다.
