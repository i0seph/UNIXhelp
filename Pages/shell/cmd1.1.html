C shell의 filename completion

filename completion 기능을 사용하려면 filec 쉘 변수가 지정되어야 합니다. 

   set filec

윗 내용을 .cshrc 파일에 추하해 주면 되겠지요.

사용법은 ESC 글쇠를 사용합니다. 

   ls -l on<ESC>line_help

이때 입력한 문자열로 시작하는 이름이 여러개 있을 경우에는 경고음만 냅니다.

그 이름이 디렉토리일 경우는 그 이름 끝에 / 문자가 나타납니다.

가능한 이름 목록

입력한 문자열로 시작하는 이름을 가진 모든 것을 보려면, CTRL 글쇠를 누르고, D 글쇠를 칩니다.

   cat co<CTRL-D>
   commands.xdh  concepts.xdh  conventions/ conver_man/
   cat co
