Bourne Again shell의 command history 사용하기

command history list(사용내역 목록)를 보려면 다음과 같이 합니다:

    history

이렇게 하면, 명령행에서 사용한 명령들이 차례로 각 번호와 함께 나타납니다. 초기값으로 여기서 나타나는 목록은 이전에 미리 저장된 command history list 입니다.

  • 다음 login을 위해 command history 저장하기
  • history list의 명령 다시 실행하기
  • history list의 명령들 미리 보기
  • 이전 명령행 편집하기
