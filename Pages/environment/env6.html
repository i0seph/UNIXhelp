자신이 사용하는 쉘 알아보기

자신이 사용하는 쉘이 어떤 것인지를 알아보려면, SHELL 환경 변수 값을 조사해 보면 됩니다. 그 방법은 다음과 같습니다. 

   echo $SHELL

이렇게 명령을 내리면 이 환경 변수의 값을 보여줍니다.

이렇게 보여주는 내용은 경로이름인데, 각 경로 명령이 뜻하는 쉘은 다음과 같습니다. 

경로이름                쉘
__________________________________________________________

/.../sh                 Bourne shell
/.../csh                C shell
/.../tcsh               TC shell
/.../ksh                Korn shell
/.../bash               Bourne Again SHell
__________________________________________________________
