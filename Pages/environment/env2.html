현재 환경 값 알아보기

한 환경 변수의 그 값을 알아 보려면, echo 명령을 사용합니다. 예를 들면:

   echo $PATH

이 명령은 PATH 변수의 값을 보여줍니다. 환경 변수는 일반적으로 대문자를 사용합니다.

$ 문자는 쉘에서 그 문자 자신이 아닌 변수를 가리키는 메타문자입니다.

현재 자신이 사용하는 쉘에서 사용하는 모든 변수 값을 함께 보는 방법도 있습니다.

Bourne 쉘에서는 이 방법으로 set 명령을 사용합니다.

C 나 TC 쉘에서는 printenv 이나, env 명령은 환경 변수와 그 값을 보여주고, set 명령은 쉘 변수와 그 값을 보여줍니다.

변수 목록이 너무 많아, 한 화면이 넘어 갈 경우에는 파이프를 이용해서 pager로 볼 수 있습니다. 예를 들면

   set | more
