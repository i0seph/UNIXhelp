쉘 변수 지정하기

임시적으로 쉘 변수를 지정하는 방법은 다음과 같습니다:

   set 변수이름=변수값

이렇게 지정한 변수 값은 그 쉘이 끝날 때까지, 다르게 바꾸지 않는 한 그대로 유지됩니다.

바뀐 변수 값을 계속해서 사용하려면 다음과 같이 합니다:

1. 문서 편집기로 .cshrc (.tcshrc) 파일을 열어서 다음과 같은 형식으로 추가합니다:

   set 변수이름=변수값

2. 이 파일을 저장하고 편집기를 종료합니다.

3. 다음 명령을 실행합니다:

   source .cshrc

이상의 작업이 끝나면, 쉘 변수값이 바뀌게 되며, 항상 C 쉘이 시작할 때, 여기서 장한 값으로 설정을 합니다.
