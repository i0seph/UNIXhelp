자주 쓰이는 환경 변수들

이름          설명



EDITOR        mail 프로그램과 같은 다른 프로그램에서 
              기본적으로 사용하는 문서 편집기 

PAGER         사용 설명서를 보는 man 같은 프로그램에서 그 내용을 
              보는 보기 프로그램

PATH          실행 파일의 경로. 쉘에서 디렉토리를 지정하지 않은
              명령은 이 변수에 지정된 디렉토리에서만 찾아 
              있으면 그것을 실행합니다.

PRINTER       lpr 명령에서 사용할 기본 프린터

SHELL         당신이 사용하는 기본 쉘

TERM          문서 편집기나, pager에서 사용할 터미날 환경

TZ            time zone(지역 시간)
