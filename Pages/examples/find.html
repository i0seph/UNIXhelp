find 명령 사용 보기

현재 디렉토리 안에 있는 한 파일을 찾는 경우:

   find . -name mtg_jan92 -print

이것은 현재 디렉토리 안에서 파일 이름이 mtg_jan92로 된 파일을 찾아 그 결과 값을 보여줍니다. 만일 없다면 아무 것도 보여주지 않습니다.

홈 디렉토리 안에 있는 파일 찾기

   find ~/ -name README -print

이것은 현재 자신의 홈 디렉토리와 그 하위 모든 디렉토리를 대상으로 README 파일 찾아 그 결과를 보여줍니다.

현재 디렉토리에의 여러 파일 찾기

   find . -name '*.fm' -print

이 명령은 현재 디렉토리에서 .fm 문자로 끝나는 모든 파일을 찾습니다.

디렉토리 찾기:

   find /usr/local -name gnu -type d -print

이것은 /usr/local 디렉토리를 대상으로 그 하위 디렉토리인 gnu 디렉토리를 찾습니다.
