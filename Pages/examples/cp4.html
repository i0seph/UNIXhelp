cp 명령 사용 보기

한 하위 디렉토리 안에 있는 파일을 현재 디렉토리로 복사 하는 경우: 

   cp notes/note3 sect3.txt

이것은 현재 디렉토리의 하위 디렉토리인 notes 안에 있는 sect3 파일을 현재 디렉토리에서 sect3 파일로 복사하는 경우입니다.

똑같은 이름으로 현재 디렉토리로 복사하는 경우: 

   cp /usr/lib/more.help .

이것은 /usr/lib/more.help 파일을 현재 디렉토리에 같은 이름으로 복사하는 경우입니다.

다른 사용자의 디렉토리 안에 있는 파일을 복사하는 경우: 

   cp ~helper/tmp/for_john tmp/comments

이것은 helper라는 사용자의 홈 디렉토리 안에 있는 tmp 디렉토리 안에 있는 for_john 파일을 현재 디렉토리의 tmp 디렉토리 안에 comments 파일로 복사하는 경우입니다.

이렇게 ~helper/tmp/for_john 같은 원본 이름을 지정할 수도 있습니다.
