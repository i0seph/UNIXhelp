vi 옵션 영구히 바꾸기 보기

.exrc 파일에 다음 내용을 추가하면,

   set ic
   set number
   set sh=/usr/local/bin/Tcsh
   set wm=5

vi 편집기의 옵션은 다음과 같이 바뀝니다:

  1. 문자열 찾기를 할 때, 대소문자 구분을 하지 않습니다.
  2. 줄 번호를 보입니다.
  3. 사용할 쉘로 tc 쉘을 사용합니다.
  4. 오른 쪽 경계값을 5로 지정합니다.

EXINIT 환경 변수를 이용한 예:

   setenv EXINIT 'set ic number sh=/usr/local/bin/Tcsh wm=5'

이것은 C나 TC 쉘 사용자가 앞의 예제에서 지정하는 것과 같은 기능을 환경 변수로 지정한 경우입니다.

.exrc 파일에서 이미 같은 변수에 대한 값을 지정한 경우는 값들은 무시되고, EXINIT 환경 변수의 값이 사용됩니다.
