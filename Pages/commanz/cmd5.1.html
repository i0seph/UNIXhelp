which 명령으로 명령어들을 찾는 보기

하나의 명령어가 어디 있는지 찾아 보는 예: 

   which ue
   /usr/local/bin/ue

MicroEMACS 편집기가 어디 있는지를 찾아보는 예입니다. 즉, 이 파일의 절대경로와 그 파일 이름은 /usr/local/bin/ue 라고 알려줍니다.

Korn 쉘이 어디 있는지 찾아보는 예: 

   which ksh
   /bin/ksh

윗 예제에서 알 수 있듯이 Korn 쉘은 /bin 경로 안에 있군요.

여러개의 명령어들을 같이 찾는 예: 

   which delete talk ps
   No delete in /usr/local/utils/bin:/bin:/usr/bin:
   /usr/local/ucb/bin:/usr/local/gnu/bin:
   /usr/local/public/bin:/usr/bin/X11:.
   /usr/bin/talk
   /bin/ps

윗 예제는 현재 PATH 환경변수 값으로 지정된 경로 안에는 delete 명령이 없음을 알려줍니다.

talk, ps 명령은 그 경로 안에 있어, 해당 경로와 이름을 보여줍니다.
