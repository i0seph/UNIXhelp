ftp 프로그램으로 파일 가져오기 보기

리모트 호스트에서 하나의 파일을 가져오는 경우:

   ftp> get news

윗 명령은 리모트 호스트 현재 디렉토리 안에 있는 news라는 파일을 로칼 호스트 현재 디렉토리로 가져오는 명령입니다. 여기서 가져온다라는 뜻은 복사한다는 뜻과 같습니다.

이때, 로칼 파일 이름을 지정하지 않으면, 리모트 호스트의 파일 이름 그대로를 사용합니다.

리모트 호스트의 하위 디렉토리에 있는 파일을 가져오는 경우:

   ftp> get reports/quarterly/march tmp/month3

이 명령은 리모트 호스트 현재 디렉토리의 하위 디렉토리인 reports/quarterly 안에 있는 march 파일을 로칼 호스트의 현재 디렉토리의 하위 디렉토리인 tmp 디렉토리 안에 month3 파일로 복사하는 명령입니다.

리모트 호스트의 여러 파일을 가져오는 경우:

   ftp> mget *.ps

이것은 리모트 호스트 현재 디렉토리 안에 있는 .ps로 끝나는 모든 파일을 로칼 호스트의 현재 디렉토리로 가져오는 명령입니다. 이때는 각 파일의 전송시 그 파일을 정말 전송할 것인지를 물어보는데, 이때, y를 누르면, 복사를 하고, n을 누르면 취소합니다.
