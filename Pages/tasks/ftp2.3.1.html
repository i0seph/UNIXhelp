ftp 프로그램으로 파일 보내기 보기

로칼 호스트 현재 디렉토리의 한 파일을 리모트 호스트 현재 디렉토리로 보내는 예제:

   ftp> send afile

윗 명령은 afile 파일을 보내는 명령인데, 여기서 보낸다는 뜻은 리모트 호스트에 복사한다는 의미입니다.

이때 리모트 호스트에 복사될 이름을 지정하지 않으면, 로칼 호스트에서 사용하는 그 파일 이름 그대로를 사용합니다.

경로이름과 파일 이름을 양쪽 다 지정한 경우:

   ftp> put REPORTS/april  reports/month4

이것은 현재 로칼 디렉토리의 하위 디렉토리인 REPORTS 디렉토리 안에 있는 april 파일을 리모트 호스트 현재 디렉토리의 하위 디렉토리인 reports 디렉토리 안에 month4라는 파일로 복사하는 명령입니다.

여러 파일을 함께 보내는 경우:

   ftp> mput *.xdh
   mput index.xdh?

이것은 로칼 호스트 현재 디렉토리 안에 있는 .xdh로 끝나는 모든 파일을 리모트 호스트 현재 디렉토리로 보내는 명령인데, 이때는 위에서와 같이 각 파일을 정말 보낼 것인지를 물어봅니다. 이때 y를 입력하면, 복사 작업을 계속 하고, n을 입력하면, 취소합니다.
