mv 명령의 사용 보기

파일 이름 바꾸기 예제:

   mv junk precious

이것은 junk라는 파일의 이름을 precious라는 이름으로 바꾸는 예제입니다.

mv 명령을 사용할 때 확인절차를 사용하는 예제:

    mv -i draft1.txt part1.txt
    remove part1.txt? n

윗 예제에서 보듯이, draft1.txt 파일의 이름을 part1.txt라는 이름으로 바꾸려고 했는데, 이미 대상 파일이 있어, 이것을 지울 것인지를 물어봅니다.

n을 입력하면, 이 명령이 취소되고, y를 입력하면, 덮어쓰겠지요.

한 파일을 다른 디렉토리로 옮기는 예제 :

    mv precious treasure

윗 명령은 precious라는 파일을 현재 디렉토리의 하위 디렉토리인 treasure 디렉토리로 옮기는 예제입니다.

한 파일을 다른 디렉토리의 다른 이름으로 옮기는 예제:

   mv jewel casket/amethyst

윗 명령은 jewel이라는 파일을 현재 디렉토리의 하위 디렉토리인 casket 안에, amethyst라는 이름으로 옮깁니다.

디렉토리를 옮겨고 이름을 바꾸는 예제:

   mv vr1.1 ../environments/virtual_rooms

윗 명령은 현재 디렉토리의 하위 디렉토리인 vr1.1이라는 디렉토리를 ../environments 디렉토리의 virtual_rooms 이름으로 바꾸어 옮기는 것입니다.
