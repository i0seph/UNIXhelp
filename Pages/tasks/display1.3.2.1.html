less 명령 사용자 정의 보기

LESS 환경 변수에 어떤 값을 지정하려면 각 쉘 별로 다음과 같이 합니다.

C shell / TC shell

파일의 크기를 보이려면: 

   setenv LESS -P%B

Bourne shell / BASH

   LESS=-P;export LESS

이렇게 하면, 보고 있는 파일의 크기를 바이트 단위로 보여줍니다.

C shell / TC shell

프롬프트를 바꾸고, 보는 방법을 바꾸려면:

   setenv LESS '-N -S -P%f'

Bourne shell / BASH

   LESS='-N -S -P%f';export LESS

이렇게 하면, 프롬프트에 파일 이름이 나타나고 (-P%f), 각 줄에 줄 번호가 보이고 (-N), 한 화면의 범위를 벗어나는 긴 줄일 경우 그것을 줄 바꿈 하지 않고, 그대로 보여줍니다(-S).

사용할 수 있는 옵션들의 자세한 설명을 사용설명서를 참조 하십시오.
