접근 권한 초기값 바꾸기

먼저 이 글을 읽기 전에, 접근 권한을 숫자로 지정하는 방법에 대한 이해가 필요할 것입니다.

파일이나, 디렉토리를 만들 때 마다, 이들의 접근 권한은 미리 지정된 값으로 정해집니다. 이 값은 파일 생성 모드 마스크에 의해서 정의됩니다.

이 마스크 값을 살펴 보려면 다음과 같은 명령을 사용합니다:

   umask

윗 명령의 결과로 다음과 같이 숫자 세개가 보여지는데:

   077

가끔 0(영)이 생략되기도 합니다. 물론 이 숫자값은 각 시스템 마다 다르겠지요.

이 숫자 값은 파일의 경우는 666, 디렉토리의 경우는 777을 빼어, 초기 접근 권한값을 파악하는데, 사용됩니다.

보기:

    777     (디렉토리 시스템 값)
   -077     (umask 값)
    ---
    700     (접근 권한의 초기값
             rwx------)

한편 이 마스크 값을 바꾸려면, 다음과 같은 명령을 사용합니다. 

    umask nnn

각각의 "n" 값은 0에서 7까지의 숫자입니다.
