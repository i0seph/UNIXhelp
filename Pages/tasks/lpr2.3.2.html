pr 명령 사용 보기

한 파일 미리 보기:

   pr chapter3.txt

이것은 chapter3.txt 파일의 인쇄 때 보여것을 미리 봅니다.

pager를 이용해서 미리 보기

   pr part2.txt | less

part2.txt 파일을 pr 명령을 통해서 미리 보는데, 그 출력을 less의 입력으로 사용해서 봅니다.

줄력 양식 바꾸어 보기

   pr -l 40 -h Using Hypertext comms_paper | more

comms_paper 파일을 보는데, 한 쪽당 40 줄로 하고, 제목을 "Using Hypertext"로 해서 파이프를 이용해 more로 봅니다.

인쇄와 연결하기:

   pr -n2 | lpr

이 명령은 표준 입력을 그 내용으로 입력받아, pr 명령으로 양식화 해서, 인쇄하는 경우입니다.
