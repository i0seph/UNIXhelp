mkdir 사용에서의 문제들

mkdir: directory_name : File exists

이것은 지정한 이름이 이미 파일로 존재한다는 것을 알리는 것입니다. 그래서, 이 명령의 인자로 디렉토리의 이름을 지정하는 뜻입니다.

usage: mkdir [ -p] directory ...

이것은 해당 상위 디렉토리가 없을 때, -p 옵션을 부여해 주면, 그 상위 디렉토리까지 생성해 줌을 알립니다.

mkdir: pathname : Read-only file system

이 디렉토리에 대해서는 하위 디렉토리를 만들 권한이 부여되어 있지 않음을 나타냅니다.
