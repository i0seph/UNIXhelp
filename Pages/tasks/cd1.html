cd 명령 사용의 보기

하나의 하위 디렉토리로 이동하는 예제: 

   cd Firstyear

윗 예제는 현재 경로에서 Firstyear라는 하위 경로로 이동하는 예제입니다.

현재 경로의 상위 경로로 이동하는 예제: 

   cd ..

윗 예제는 숨은 디렉토리인 .. 디렉토리로 이동하는 명령인데, .. 이름은 상대 경로로 현재 디렉토리의 바로 상위 디렉토리를 뜻합니다.

이렇게 짧게 사용할 수 있는 이름은 경로 이동을 빠르게 하는데, 유용하게 사용됩니다.

다른 상대 경로로 이동하려면 다음과 같은 형식을 취합니다:

   cd ../Secondyear

윗 예제는 현재 경로의 상위 경로에 대한 Secondyear라는 하위 경로로 이동하는 명령입니다.

절대 경로를 대상으로 이동하는 예제: 

   cd /home/physics/ercy04/ProjectX

윗 예제는 홈 디렉토리의 ercy04 라는 사용자의 ProjectX라는 디렉토리로 이동하는 명령입니다.

자신의 홈 디렉토리로 이동하려면 다음과 같이 합니다:

   cd
