리모트 호스트 연결 보기

리모트 호스트의 ftp 서버에 연결하는 예제:

   ftp> open castor
   Connected to castor.ucs.ed.ac.uk.
   220 castor FTP server (SunOS 4.1) ready.
   Name (castor:erpl08): erpl08
   331 Password required for erpl08.
   Password: not echoed
   230 User erpl08 logged in.
   ftp>

윗 예제는 "castor"이라는 리모트 호스트에 접속하는 과정을 보여주고 있습니다. 이때, username, password는 그 호스트에 접속 가능한 것이여야겠지요.

리모트 호스트의 "anonymous ftp" 서버에 접속하는 예제:

   ftp> open info.cern.ch
   Connected to nxoc01.cern.ch.
   220 nxoc01 FTP server (Version 5.20 (NeXT 1.0) .....
   Name (info.cern.ch:erpl08): anonymous
   331 Guest login ok, send ident as password.
   Password: enter your email address here
   230 Guest login ok, access restrictions apply.
   ftp>

윗 예제는 스위스 제네바에 있는 "유럽 미분자 물리학 도서관" 이라는 "info.cern.ch" 익명 ftp 서버에 접속하는 과정을 보여줍니다.

여기서는 World Wide Web에 관한 각종 소프트웨어와 문서들을 구할 수 있습니다.
