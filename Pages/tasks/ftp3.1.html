호스트에 대한 정보 얻기

이미 호스트 이름을 알고 있다면, hosts 명령으로 NRS(Name Registration Scheme) 안에 나타나는지와 어떤 서비를 제공하는지를 확인해 볼 수 있습니다.

JANET 네트워크에서의 호스트 이름은 다음과 같은 형식입니다.

   uk.ac.기관이름.호스트이름

예제

   hosts festival

윗 명령은 다음 정보를 보여줍니다:

   Standard Name    : uk.ac.edinburgh.festival
   Abbreviated Name : uk.ac.ed.festival
   Description      : EUCS Central Unix Service
   Services         : X29/JANET FTP/JANET MAIL/JANET

위에서 보듯이 이 호스트에서 FTP/JANET 서비스를 제공함으로 로칼 호스트에서 이곳을 리모트 호스트로 해서 자료를 전송할 수 있습니다.

UK PSS (Packet Switch Stream)에서의 상용 호스트들의 이름은 다음과 같은 형식입니다:

   uk.co.기관이름.호스트이름

예제:

   hosts ixi

윗 명령은 다음 정보를 보여줍니다:

   Standard Name    : uk.co.ixi-limited
   Abbreviated Name : uk.co.x
   Description      : IXI limited
   Services         : MAIL/JANET MAIL/PSS
   JANET MAIL Relay : uk.ac.uknet
   PSS MAIL Relay   : uk.ac.uknet

위에서 보듯이 IXI Ltd에는 전자 편지를 보낼 수 있으나, 파일 전송은 하지 못합니다.
