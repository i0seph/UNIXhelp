파일 이름을 명령행에서 사용하는 경우

사용법은 다음과 같습니다.

   ue 파일이름

가령, "편지"라는 새로운 파일을 만든다면, 다음과 같이 합니다. 

   ue 편지

이 명령이 실행되면, 화면 아래쪽에 다음과 같은 내용이 보입니다. 

== MicroEMACS 3.9e() == letter == File: 편지 ========

윗 구성의 설명은 다음과 같습니다:

  1. 먼저 MicroEMACS 이름과 그 버전 번호
  2. 다음 편집 모드 이름
  3. "버퍼"라고 하는 컴퓨터 측 편집 공간 이름
  4. 마지막으로 사용자가 지정한 파일이름

대부분의 emacs 편집기가 화면 아래쪽이 이런 비슷한 정보를 보입니다. 각 편집기의 구체적인 내용을 살펴보십시오.
