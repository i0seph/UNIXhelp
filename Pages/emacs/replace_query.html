물어보면서 바꾸기

이 기능 또한 문서 전체에 걸쳐 바꾸기를 하지만, 이때는 해당 문자열이 발견될 때 마다, 사용자에게 어떻게 할 것인지를 물어봅니다.

MicroEMACS 편집기 경우:

1. M-^R 입력 

   Query replace []<META>:

2. 찾을 문자열을 입력하고, META 글쇠 침. 

   with []<META>:

3. 바꿀 문자열을 입력하고, META 글쇠 침.

여기까지 하면, 찾을 문자열이 처음 발견 되는 곳으로 커서가 이동되고 다음과 같은 형식으로 사용자에게 물어봅니다. 

   Replace 'junk' with 'test'?

4. 이때 다음 중 한 문자를 입력합니다:

    y  (yes, 오냐, 바꾼다.)

    n  (no, 아니, 안바꾼다.)

    !  (묻지 말고 니 혼자 해!)

    ^G (그만 하거라.)

hanemacs 편집기 경우:

"M-x query-replace" 기능을 이용합니다. (자세한 이야기는 GNU emacs의 info 파일을 보세요.)
