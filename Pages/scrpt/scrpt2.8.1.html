디버깅 보기:

   cat example
   #!/bin/sh
   TEST1=result1
   TEST2=result2
   if [ $TEST1 = "result2" ]
   then
     echo $TEST1
   fi
   if [ $TEST1 = "result1" ]

example 스크립트에서는 test3 변수가 정의되어 있지 않기 때문에, 다음의 오류를 보입니다. 

   example
   result1
   [: argument expected

이 스크립트가 어떻게 해서 오류가 났는지, 확인해 보기위해, -x 옵션을 사용한 결과는 다음과 같습니다:

   sh -x example
   TEST1=result1
   TEST2=result2
   + [ result1 = result2 ]
   + [ result1 = result1 ]
   + echo result1
   result1
   + [ = whosit ]
   example: [: argument expected
