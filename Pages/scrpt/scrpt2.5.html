test 명령 사용하기

test 명령은 주로 if 문에서 판단을 해야할 경우에 사용되는데, 자세한 사항은 test 사용자 설명서를 참조하세요. 예를 들어:

   if test ! -f $FILE
   then
     if test "$WARN" = "yes"
     then
       echo "$FILE 파일이 없습니다"
     fi
   fi

윗 스크립트는, FILE 변수에 지정된 변수값에 사용하고자 하는 파일 이름을 지정하고, 그것이 실제로 있는지 검사하고, 없다면, 다시, WARN 변수 값이 yes 인지를 검사해서, 그렇다면, 파일이 없다는 메시지를 보이는 것입니다.
