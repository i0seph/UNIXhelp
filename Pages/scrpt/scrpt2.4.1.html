if 문

if 문은 일반적으로 스크립트 안에서 한 명령을 실행시키고 그 명령의 리틴값을 살펴서 그에 따라 적당한 작업을 할 수 있도록 합니다. 일반적인 문법은 다음과 같습니다:

   if test
   then
      commands(참일 때)
   else
      commands(거짓일 때)
   fi

then, else, fi 단어들은 쉘에서 사용되는 예약어이며, 이것들의 구분은 그 단어 다음에 줄바꿈문자를 사용하든지, ; (세미콜론) 문자를 사용합니다. if 문의 끝은 fi로 끝냅니다.

if 문은 다음과 같이 내포문으로 사용될 수도 있습니다:

   if ...
   then ...
   else if ...
       ...
     fi
   fi

위의 else if 문을 간단히 줄여서, elif 문으로 사용할 수도 있습니다:

   if ...
   then ...
   elif ...
     ...
   fi
