특별한 쉘 변수

다음은 쉘에서 특별한 의미로 사용되는 변수 이름들입니다:


이름          설명


$1 - $9       각 명령행 인자.

$0            쉘 스크립트 자체, 또는 그 명령 자체.

$#            명령행에서 주워진 명령행 인자의 갯수.

$?            가장 최근에 마친 명령의 리턴값이 10진수 문자열로 
              저장되는 변수. 보통 한 명령이 아무 오류 없이 
              성공적으로 끝나면, 이 값은 0 (숫자 영)이고, 
              그렇지 않으면, 0 이외의 값이 지정됩니다.

$$            현재 쉘의 프로세스 번호 - 보통 고유한 파일이름을
              만드려고 할 때 사용됩니다.

$!            현재 background로 실행되는 마직막 명령의 
              프로세스 번호

$-            현재 쉘이 호출될 때 사용한 옵션들

$*            명령행 인자의 모든 것.

$@           윗 변수와 똑같지만, 따움표 처리 문제에서 다름.

참고

$* 변수와 $@ 변수의 차이점은 각각의 실재 해당 인자를 처리하는 방식에 있습니다.

"$*" 이렇게 변수가 따움표로 묶겨지면, 쉘에서의 공백문자 식별자(IFS 변수값)로 무조건 처리하는 것으로 '1 2' 3 같은 명령행 인자가 있다면, 이것을 "1 2 3" 이렇게 하나의 인자로 처리합니다.

"$@" 이렇게 변수가 따움표로 묶겨지면, '1 2' 3 같은 명령행 인자가 있다면, 이것을 "1 2" " 3" 이렇게 두개의 인자로 처리합니다.
