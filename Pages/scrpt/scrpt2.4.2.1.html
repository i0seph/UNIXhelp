&& 사용 보기

이전 명령의 실행 상태에 따라, 다음 명령 실행시키기:

   cat deliver
   #!/bin/sh
   # 사용법: deliver username  파일
   { cat $2 | write $1 ; } && echo 작업완료

윗 스크립트를 다음과 같이 사용한다면:

   deliver keith greeting

먼저 { cat $2 | write $1 ; } 부분을 실행합니다. 그래서, greeting 파일을 cat 명령으로 표준 출력으로 보내어, 그것을 파이프를 통해, write 명령의 표준 입력으로 사용하고, 그결과 keith 사용자에게 그 내용이 보여지겠지요.

여기서 명령행 인자변수들인 $1, $2 변수들을 사용한 것과, 파이프 기능의 사용이 거기서 끝남을 알리는 ; (세미콜론) 문자가 사용되었음을 참고하십시오.

이때, 첫번째 명령이 성공이 성공적으로 끝나면, 두번째 명령이 실행되어, 작업완료라는 메시지를 결과적으로 표준출력으로 보이겠지요.
