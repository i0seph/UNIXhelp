보내기 모드에서 사용할 수 있는 명령들

다음은 편지 내용을 쓰는 상태에서 사용할 수 있는 명령들입니다. 즉, 그 명령을 그대로 입력하고 엔터를 치면 그 명령이 실행 됩니다. 


설명 ...                                          명령



편지 쓰기 취소 					~x

편지 내용 보기					~p

사용자 추가 Cc: list				~c 사용자

사용자 추가 Bcc: list				~b 사용자

도움말 보기					~?

EDITOR 환경 변수에 지정된 편집기 호출 		~v
	
파일을 편지 내용으로 끼워넣기 			~r 파일이름
	
서명(signature) 추가하기                        ~a 또는 ~A
	
사용자 추가 To: list                            ~t 사용자
	
쉘 명령 사용 					~! 쉘명령
	
쉘 명령의 출력 내용을 끼워넣기 			~<! 쉘명령
